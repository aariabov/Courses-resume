# Тестовые сценарии для проверки работоспособности

## Оплата подписки

### Аноним
- вход на меню
- получение userInfo - null
- subscribe - видит цену, при нажатии Оформить открывает форму регистрации
- регистрация и логин

### Юзер залогинен
- получение userInfo после регистрации и логина
- изменилось меню
- Моя подписка - нет активной подписки, история недоступна
- Кнопка оформить - переход на subscribe
- Оформить - сохранение платежа в бд, переход на страницу оплаты
- Отказ от платежа - переход на страницу результат платежа - что-то пошло не так
- Попробовать снова - переход на subscribe
- Оформить - сохранение платежа в бд, переход на страницу оплаты

### Юзер с подпиской
- Оплата - переход на страницу результат платежа - Успешно, есть кнопка Моя подписка, обновилась иконка
- История платежей - показ истории платежей
- Моя подписка - есть активная подписка
- Выполнить упражнение - ок
