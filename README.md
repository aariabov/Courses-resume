# Проект с курсами и упражнениями по C#

## Описание
Сайт содержит курсы, статьи, упражнения с автоматической проверкой и имитацию консольного приложения.

При выполнении упражнения, на сервере запускается Docker-контейнер, прогоняются тесты, результат читается из потока, затем обрабатывается и отправляется пользователю.

## Описание solution

- Common - общая сборка
- Devpull - основной веб-проект
- ClientCodeRunner - проект, предназначенный для выполнения клиентского кода
- ExerciseTester - проект, предназначенный для проверки упражнений
- UnitTests - проект с юнит-тестами
- IntegrationTests - проект с интеграционными тестами
