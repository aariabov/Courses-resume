# Шаг 2. Вывод игрового поля

На этом шаге изучим немного теории, познакомимся с выводом в консоль, переменными и массивами. И приступим к написанию игры Крестики-нолики.

На шаге будет рассмотрено:

- вывод в консоль
- типы данных
- объявление переменных
- чтение переменных
- изменение переменных
- интерполяция
- массивы
- вывод игрового поля
