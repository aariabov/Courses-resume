# Шаг 4. Ходы игроков

На этом шаге изучим логический тип данных, ветвление и познакомимся с циклами. Также будут реализованы ходы игроков по очереди.

На шаге будет рассмотрено:

- логический тип данных
- логические операции
- ветвление
- цикл for
- перебор элементов массива
- использование оператора break
- использование оператора continue
- реализация ходов игроков по очереди
