# Шаг 1. О языке C#

На первом шаге вы установите IDE, создадите консольное приложение и познакомитесь с языком C#. Будут рассмотрены его основные характеристики и области применения. Вы также узнаете как писать комментарии и как C# работает с ошибками.

На шаге будет рассмотрено:

- обзор IDE
- установка IDE
- создание консольного приложения
- пример простой программы на C#
- основные характеристики языка
- области использования
- комментарии
- типы ошибок
- поиск и исправление ошибок
